## Getting Help

### Login Issues
- Make sure you're using the correct email and password
- Passwords are case-sensitive
- Try refreshing the page

### Navigation
- Use the sidebar menu to navigate between sections
- Click your role name in the top right to logout
- The logo takes you back to your dashboard

### Features
- Each section has its own purpose - explore the sidebar
- Look for buttons like "View Details" or "Start Activity"
- Progress bars show completion status

## Tips for Success

### For Admins:
- Regularly check the reports section for insights
- Keep user information up to date
- Monitor system activity

### For Staff:
- Review student progress weekly
- Coordinate with coaches on student needs
- Track program completion rates

### For Coaches:
- Add detailed session notes after each meeting
- Review student progress before sessions
- Keep resource materials handy

### For Parents:
- Check your teen's progress regularly
- Read session notes from coaches
- Use the resources section for guidance

### For Teens:
- Complete daily activities to earn points
- Use your journal to reflect on growth
- Check your achievements to see progress
